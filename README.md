# WeatherApp

WeatherApp – aplicație web construită cu Java Spring Boot și un frontend simplu în HTML/CSS. Permite utilizatorului să introducă un oraș și să primească date meteo în timp real (temperatură, vânt, umiditate etc.), folosind OpenWeatherMap API.  
**Demo live:** (prima încărcare poate dura 30–50 secunde)

## Features
- Introducere oraș pentru obținerea vremii
- Vizualizare temperatură, umiditate și viteza vântului
- Interfață simplă și responsive
- Răspuns rapid folosind OpenWeatherMap API

---

## Technologies
- Java Spring Boot (backend)
- HTML / CSS / JavaScript (frontend)
- OpenWeatherMap API
